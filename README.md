# aFetch ~ Another Fetch
Simple system info written in C

**Requirments**
*  A /etc/os-release file

**Currently Supports**
*  void

**Partial Support (No logos yet)**
*  Arch
*  Gentoo (not sure about the package manager)
*  Fedora
*  FreeBSD (not sure about the package manager)
