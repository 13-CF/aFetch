# aFetch ~ Another Fetch
Simple system info written in C

**Requirments**
*  A /etc/os-release file (/var/run/os-release on BSDs also works)
*  A C compiler

**Currently Supports**
*  Void
*  Gentoo
*  Arch
*  Artix
*  Fedora
*  FreeBSD

**Please request your distro if you want it added**
